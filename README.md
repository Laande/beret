# Beret

Walk to the furni when its the good color

# Commands

`/tile` -> Double click to the furni

*saves x;y and id*

`/color x` -> Replace X with the number of the furni ( like yellow = 1, red = 3 )

*/color 2*

`/reset` -> Reset the extension

*clear id, x;y, color*
